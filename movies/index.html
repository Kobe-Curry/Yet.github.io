2021-10-17 / 2002 美国 犯罪 经典 莱昂纳多·迪卡普里奥 / ★★★★★ 力荐
剧情非常精彩，小李子扮演的Frank在剧中是一名智商超群的罪犯，生活里扮演着各种职业的角色，到处骗吃骗喝骗色，一路凭借自己的姿色和智慧蒙混过关，最后还能与一名护士结婚，不过还是被Carl探员给抓了。意想不到的是电影由真人真事改编，这人真是钻了科技不发达空子的绝世奇才啊。整部影片让我感触颇深的地方是，Frank虽然通过行骗过上了自己想要的生活，但每个平安夜还是会打电话给Carl探员，表面上是刺激和嘲讽他的抓捕能力，实则Frank很孤独，在外面没有真正意义上的亲人，父母早已离异，周围也都是随时可能会分道扬镳的表面朋友，而此时的Carl似乎嗅到了Frank的弱点，并捧腹大笑起来，他们亦敌亦友，世上除了亲人外也只有Carl更重视和了解Frank了。电影结局是好的，Frank出狱后，由于杰出的识别假钞能力当上了FBI的技术顾问

过把瘾
2021-10-08 / 1994 江珊 王志文 经典 爱情 / ★★★★★ 力荐
这应该是我目前看过最好看的爱情电视剧，没有之一，万万没想到1994年的老剧如此吸引着我。首先，里面所有的台词没有一句废话，句句都很接地气，非常真实，剧情也不拖泥带水，虽短短8集，但足够说事了，反观现在的电视剧，唉。其次，三位女演员是真的很美，那种没有滤镜的天然美，要我出生在那个年代，一定会喜欢江珊这种类型的。再夸夸剧情吧，虽然有点狗血，但不知道为什么我居然很喜欢看方言杜梅之间的顶级拉扯，太有意思了。最后，该剧每个演员都有Ta存在的道理，并没有说因为Ta是配角，就不重要。总之，太好看了，也学到了两人相爱过程中要注意的点，再说句最喜欢的台词：“多清楚的字，离近点看还得花”

夜·店
2021-10-01 / 2009 喜剧 犯罪 搞笑 黑色幽默 / ★★★★★ 力荐
记得八九年前看过，现在重温依然觉得非常好笑，顺便说句我发现电影中的两位主角已去世，再次感慨时间过得真快。话说回来，电影的幽默桥段非常多，安排的非常紧凑，且有节奏，它不像传统电影那样递进、高潮、结尾，而它是全程发生着一个接一个离谱但又合理的搞笑事情。在这里不得不佩服编剧的脑洞，既能展现电影幽默感，又能使每个演员之间能推动情节的发展。最后结尾电影把男店员对女店员的暗恋进行了升华，虽然有点生硬，但我觉得问题不大，不算是败笔，结局是he就行。总之，电影非常搞笑，值得放松时去看！

菊次郎的夏天 / 菊次郎の夏
2021-08-19 / 日本电影 北野武 久石让 童年 温情 / ★★★★☆ 推荐
整体观感就一个词——纯粹，它没有尘世的喧嚣，职场的勾心斗角，满眼皆宁静、纯真与怪诞，电影整体节奏较慢且不失幽默，令我非常放松，数周的IETLS自学属实令我有些疲惫，这部轻松的影片，仿佛勾起了我童年的回忆，有着很多快乐，它很简单，也很无聊，可就是这些快乐是孩子区别于大人的地方，感谢这部作品，让我重温了旧时光
盛夏未来
2021-08-03 / 校园 青春 爱情 / ★★★★☆ 推荐
31号晚观后，发现和期待的情节不太一样，《盛夏未来》前期宣传的非常甜，疯狂炒cp，可剧情里的男主确是是个实实在在的“渣男”，通过其他人的影评与看法大概了解到郑宇星可能是个同志，只是陈正道导演采用隐晦的手法来表达，让观众有所遐想，这不得不让我想到了《秘密访客》里的汪先生，刚百度完发现该电影的导演也是陈正道，迷之巧合。比较生气的是在bg里混bl，而且女主还是我最喜欢的枫妹，妹妹的荧幕第二吻换来的是冷冷地一句：“要是我也喜欢你就好了”，我直接想钻进荧幕里给他爸的来一拳。不过U1S1，两人的演技都非常优秀，不愧是北电学生，电影前半段有很多幽默桥段，台词也是非常贴近校园生活，非常的真实，出乎我意料，比较有看点。然后结尾是开放式结局，是he还是be一千个人心中有一千个哈姆雷特，虽说俩人最后没有在一起，但至少生活重回正轨

小千一家子误入仙境，无人的小镇夜晚流光溢彩、灯火通明；仙气氤氲，为汤屋蒙上一层面纱；零星几趟电车在水面上泛起圈圈涟漪；小千漫步在碧海蓝天下的绿坡；无论白天还是黑夜，这里都美如梦境。各种弦乐声交织在一起，不得不感慨下久石让先生那忧伤凄美的配乐，如听仙乐耳暂明，画面和音乐的美是全片给我最直观的感受；其实千寻和白龙之间的感情是典型的宫崎骏式爱情，还记得离别之际白龙那句：“我只能送你到这里了，剩下的路你要自己走，不要回头。”。整体观感很棒

你的名字。 / 君の名は。
2021-07-06 / 日本 治愈 爱情 青春 / ★★★★★ 力荐
《君の名は。》所拥有的一切，不只是画风的唯美，不只是剧情的迷人，不只是错位时空的遗憾，不只是两个人相遇的故事，而是由这所有的一切共同构造。那触人心底的情感流动，宛如那颗划破夜空的美丽彗星一般闪耀着。那一瞬，唯美的视觉冲击，配合着悦耳的钢琴乐，这一切都将我深深吸引，还有三叶剪短发和穿上和服后的那个模样，仿佛波及到我内心深处的审美，霎那间嘴角微扬。影片结尾，俩人看起来要再次错过，但最终都鼓起勇气同时回首问出那句：我好像在哪里见过你。伴随着樱花的落下，影片结束，最后我想说，愿青春不再遗憾～

我的姐姐
2021-04-05 / / ★★★★★ 力荐
在这部电影中，从安然的角度我多少能感受到作为姐姐所遭受的困境。现实生活中，我也有个姐姐，记得小的时候经常和她打架疯闹，她觉得我老是不听她的话，我又觉得她老欺负我、吓唬我，把我当狗腿，hh，这些都成为过去式，现在大概六七年没有一起待过半个月以上，多少有点怀念小时候无忧无虑玩耍的时光。回到这部电影，它突出了重男轻女的现实问题，其实生活中我多少能感受到重男轻女的存在，想到小时候和姐姐争爸爸妈妈更爱谁的问题就感到特别好笑，其实无论是男是女，都值得长辈的呵护与关怀，这样家庭才会和睦，子女才会孝顺。最后，我已经按捺不住要夸了，子枫妹妹的演技一如既往的自然、高超，我对短发妹妹毫无抵抗力，一个字，酷！期待今年她的另外3部电影！
8英里 / 8 Mile
