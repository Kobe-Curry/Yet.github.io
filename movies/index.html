秘密访客
2021-05-02 / 悬疑 剧情 推理 / ★★★★☆ 推荐
这种类型的电影我看的少，这回看的半懂不懂，整部剧给我直观感受就是压抑，这是内心有多黑暗的人才能写出的剧本？最后，还是得走下程序，吭吭，子枫妹妹演技一如既往的自然，答应我下次不要再吓我了好吗？
歌舞青春 / High School Musical
2021-04-24 / 美国 青春 校园 歌舞剧 / ★★★★★ 力荐
这是部非常青春的歌舞剧，我非常喜欢男主女主的和声。剧中女主是个典型的理科学霸，男主是个风云校园的篮球高手，俩人机缘巧合地在一次party上合唱，突然就开启了双方内心那部分对音乐的渴望。整体故事情节比较简单轻松，故事多了几分理想色彩，然而足以让我沉浸在美式高中歌舞剧的氛围里。观影后还有有一发现是，电影里的配乐和王力宏的音乐作品风格很类似，可能他早期受西方流行乐的影响吧，这使得我对王力宏的欣赏又多了几分，欣赏源于我的审美和对事物认知度的提升，这也是为什么他能被称为华语乐坛领军人物，哈哈扯远了。
我的姐姐
2021-04-05 / / ★★★★★ 力荐
在这部电影中，从安然的角度我多少能感受到作为姐姐所遭受的困境。现实生活中，我也有个姐姐，记得小的时候经常和她打架疯闹，她觉得我老是不听她的话，我又觉得她老欺负我、吓唬我，把我当狗腿，hh，这些都成为过去式，现在大概六七年没有一起待过半个月以上，多少有点怀念小时候无忧无虑玩耍的时光。回到这部电影，它突出了重男轻女的现实问题，其实生活中我多少能感受到重男轻女的存在，想到小时候和姐姐争爸爸妈妈更爱谁的问题就感到特别好笑，其实无论是男是女，都值得长辈的呵护与关怀，这样家庭才会和睦，子女才会孝顺。最后，我已经按捺不住要夸了，子枫妹妹的演技一如既往的自然、高超，我对短发妹妹毫无抵抗力，一个字，酷！期待今年她的另外3部电影！
8英里 / 8 Mile
2021-03-31 / 音乐 rap 美国 / ★★★★☆ 推荐
Look, if you have one shot, or one opportunity, to seize everything you ever wanted in one moment. Would you capture it or just let it slip? 很喜欢这句话。不过整部电影和我预想的不太一样，我个人比较期待的情节是Rabbit被嘲笑后发奋图强，每天如何如何练自己的battle能力，最后在一场battle上大胜，一洗前耻，虽然这有点老套。而整部电影里，兄弟情、爱情、亲情部分描绘的较多，凌晨看有点犯困。我突然发现外国电影很喜欢加上“关系非常好的主演们突然因为某事绝交，到最后又因为某事感动和好如初”的情节，有点好笑。
怦然心动 / Flipped
2021-03-30 / 初恋 青春 / ★★★★★ 力荐
你对我一见钟情，我早已看出；我对你怦然心动，你早已知晓；可最后依然没有面对面一起表达出来。有时候保持一定的神秘感、新鲜感、微妙感，青春期的爱情还是挺令人向往的，cuz all I need is jus being with u。电影的情节从两个小孩子的视角展开，讲述俩人对彼此之间的感觉和心路历程，揭秘般地让观众理解情节并能够产生共鸣。最后，珍惜身边愿意默默守候你的那个Ta吧，祝余生都能收获幸福！

少年的你
2019-10-26 / 校园霸凌 青春 校园 爱情 / ★★★★★ 力荐
这是部很棒的影视作品，由于它以校园霸凌为题材，所以整个作品基调会显得很压抑沉重，整个观影过程都很心堵、心塞。演技方面，俩演员都非常棒，张弛有度，爆发力强，前半段令我惊讶的是第一次看到千玺撩的一面，很酷很自然，后半段留下全场最佳台词：“你保护世界，我保护你”。电影制作上，画面构思巧妙，BGM配的到位，扣人心弦，最后瑞思拜曾国祥导演，期待之后的影视作品。
