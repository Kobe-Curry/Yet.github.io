2021-03-252021-04-22💻实用教程1k3分钟

大多数教程、文章都会使用下列英文通用参数，方便表达含义，你我之间的暗号，懂的都懂。

部分教程的通用参数会直接使用中文

教程&笔记常量申明

参数名释义
<UserName>项目托管平台注册时的用户名（包括 GitHub、Gitee 等）
<BlogRoot>博客本地根目录
<RepoGit>远程仓库 Git 地址
<RepoName>远程仓库名称
<OriginRepoGit>上游仓库 Git 地址
<BranchName>仓库分支名称
<Custom>自定义内容
<FilePath>文件路径，有时可能会加上文件名的后缀。
例如：<FilePath_520.jpg>，意为 520.jpg 图片的路径
<Copy>复制内容
<Email>邮箱号
<URL>链接
<> 👉 []意为可选参数
Git 三连git add .
git commit -m ""
git push origin
Hexo 三连hexo clean
hexo g
hexo s / hexo d

本站通俗易懂的介绍

我猜本站的大部分访客是 Butterfly 博客主题的圈内人士，且多数访客的专业应该和计算机相关。考虑到小部分访客是通过其它途径访问进来的，在此我想用通俗易懂的话语给你们介绍下本站

本站的大部分源代码是一位优秀博主 Jerry 作者写的，我所做的主要工作就是使用该开源博客框架搭建网站、上传自己的文章，其次是按照自己的想法与灵感通过写代码来美化网站的外貌。其次想说，我就是整个网站的主人啊，并非在某平台注册登录才能使用的那种模式，这就意味着我可以任意操纵这个网站，想发什么内容就发什么，没有人来审查你的文章是否有违禁内容，但提一句，人还是要用道德来约束自己，不能胡作非为哈哈。

页面布局介绍

本站的大部分页面布局是由 5 个大版块构成：导航栏、主体部分、组件板块、功能按钮、页脚

  • 导航栏就是最上方那一长栏。有回到首页按钮、各大功能页面的快捷入口、搜索框、深浅模式切换按钮

  • 主体部分就是文章内容或者功能页面的内容。文章里边还内嵌评论板块

  • 组件板块就是右侧若干小卡片。有博主信息卡片、文章更新列表、最近访客评论、文章目录等等

  • 功能按钮就是最右边的那几个圆圆按钮。有阅读模式、简繁切换、直达评论区等等

  • 页脚就是页面底部的那一长栏。有各大页面的快捷入口，和一些网站信息展示

主体部分的页面又分为文章页面和功能页面

  • 文章页面就是我写的文章嘛，有我的学习笔记呀、经验分享等等内容
  • 功能页面就比如标签、分类、时光机，可以快速定位到你想看的文章；再比如歌单、相册、哔哔、影评，相当于是博主我信息的一种展示；还有画廊、八音盒、网址导航，你可以理解为是一种网络工具；还有很多页面喜欢就点进去看，就不一一介绍（码字好累 Orz）

那怎么使用呢？

那肯定就是用眼睛看我写的文章内容嘛，也可以同时听听左下角音乐播放器里我收藏的音乐。

然后，阅读时可以先浏览一遍本站的版权协议，评论时可以先浏览一遍本站的隐私协议；

阅读时遇到一些难以理解的问题请：

  • 先百度
  • 也欢迎在直接文章页面下方的评论区留言
  • 与文章内容无关的问题可以点击上方导航栏中的留言信枫进行留言
  • 还可以直接给我发送邮件
