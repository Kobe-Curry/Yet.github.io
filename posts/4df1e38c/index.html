字符串

字符串运算符

前提：a = “Hello”, b = “Python”

操作符描述实例
+字符串连接a + b 输出结果： HelloPython
*重复输出字符串a*2 输出结果：HelloHello
[]通过索引获取字符串中字符a[1] 输出结果 e
[ : ]截取字符串中的一部分，遵循左闭右开原则a[1:4] 输出结果 ell
in成员运算符 - 如果字符串中包含给定字符则返回 True‘H’ in a 输出结果 True
not in成员运算符 - 如果字符串中不包含给定的字符返回 True‘M’ not in a 输出结果 True

字符串的截取和连接

字符串的常见操作

函数描述
len(string)返回字符串长度
join(seq)以指定字符串作为分隔符，将 seq 中所有的元素(的字符串表示)合并为一个新的字符串
lower()转换字符串中所有大写字符为小写
upper()转换字符串中的小写字母为大写
lstrip()截掉字符串左边的空格或指定字符
max(str)返回字符串 str 中最大的字母
min(str)返回字符串 str 中最小的字母
rstrip()删除字符串字符串末尾的空格

列表 List

列表的循环遍历

函数

函数描述
len(list)列表元素个数
max(list)返回列表元素最大值
min(list)返回列表元素最小值
list(tuple)将元组转换为列表

方法

方法描述
list.append(obj)在列表末尾添加obj
list.count(obj)统计obj在列表中出现的次数
list.index(obj)查找列表中值为obj第一个匹配项的索引位置
list.insert(index, obj)将obj插入列表下标为index
list.pop(index)移除下标为index的元素，并返回该元素的值
list.remove(obj)移除列表中值为obj的第一个匹配项
list.reverse()反转列表中元素
list.sort(key = None, reverse= False)对原列表进行排序（见下）
list.clear()清空列表
list.copy()复制列表

sort() 方法：key可以指定一个排序方法；reverse = True 降序， reverse = False 升序（默认升序）。

输出结果：降序列表：[(1,3), (2,2), (4,1)]

函数

函数描述
len(tuple)计算元组元素个数
max(tuple)返回元组中元素最大值
min(tuple)返回元组中元素最小值
tuple(list)将列表转换为元组

字典 Dictionary

根据键访问值

在我们不确定字典中是否存在某个键而又想获取其值时，可以使用get方法，还可以设置默认值

删除字典元素

常用操作

前提：dict = {'Name': 'hassan', 'Age': 20, 'Class': 'three'}

常用操作描述
print(dict[‘hassan’])通过key来访问元素并打印，不存在会抛出异常
print(dict.get(‘hassan’))通过get方法来访问元素，不存在返回None，不抛出异常
for key in dict:
print(key, dict[key])		遍历字典1
for key, val in dict.items():
print(key, val)		遍历字典2
print(dict.keys())获取所有key
print(dict.values())获取所有value
dict[‘Age’] = 21新增键值对或修改val
del dict[‘hassan’]删除元素
dict.pop(‘hassan’)将元素弹出
‘hassan’ in dict判断元素是否存在
dict.clear()清除字典内的元素
len()获取字典长度
max()获取最大的key
min()获取最小的key
