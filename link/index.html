我的友链

若贵站友链显示方式不是 flexcard，或无需网站截图，则可以将siteshot这一参数省略

本站友链添加方式

参照以下格式留言即可。如果你是 👇

比如：网址导航，文档网站等等

注意：

  1. 参数记得要与冒号之间空一格
  2. 个人搭建的博客网站我会放在本友链页，其他类型网站我会添加到我的网址导航页面
  3. 你的博客最好配置有sitemap.xml，因为本友链页会有爬虫来爬取文章进而生成朋友圈

站点截图方式 - siteshot

建议使用 thum.io 提供的 API 以获取贵站的截图

方法一：先访问以下链接获取 API 提供的截图，再将图片上传至你的图床，最后将链接填入 siteshot。

方法二：自行截图或用其它能代表贵站的图片，再将链接填入 siteshot。（要求：400✖266 像素 或 长宽比例 1.5）

方法三：直接省略（API 会自动截图，虽说 API 目前相当稳定，但本站不能保证以后也稳定）
